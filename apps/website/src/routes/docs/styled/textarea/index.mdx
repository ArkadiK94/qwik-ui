# Textarea

Displays a form textarea field or a component that looks like a textarea.

<Showcase name="hero" />

## Installation

```sh
qwik-ui add textarea
```

## Usage

```tsx
import { Textarea } from '@/components/ui/textarea';
```

```tsx
<Textarea />
```
