# Checkbox

A control that allows the user to toggle between checked and not checked.

> The current implementation of the styled checkbox uses the native html `<input type="checkbox" />` element. We intend to introduce a headless checkbox with intermediary states and better handling of colors, animations, etc. in the future. In the meantime, this native element is still better than nothing ðŸ™ƒ.

## Installation

### Run the following cli command or copy/paste the component code into your project

```sh
qwik-ui add checkbox
```

```tsx
import { PropsOf, component$ } from '@builder.io/qwik';
import { cn } from '@qwik-ui/utils';

export const Checkbox = component$<PropsOf<'input'>>(({ name, ...props }) => {
  return (
    <input
      type="checkbox"
      {...props}
      class={cn(
        'peer peer h-4 w-4 shrink-0 border-primary text-primary accent-primary ring-offset-background focus:ring-ring focus-visible:ring-ring disabled:cursor-not-allowed disabled:opacity-50 data-[state=checked]:bg-primary data-[state=checked]:text-primary-foreground',
        props.class,
      )}
      name={name}
      id={name}
    />
  );
});
```
