# Radio Group

Visually or semantically separates content.

> The current implementation of the styled radio-group uses the native html `<input type="radio" />` element. We intend to introduce a headless radio-group with better support for colors, animations, etc. In the meantime, this native element is still better than nothing ðŸ™ƒ. You can remove the accent-primary class from the RadioGroupItem component if you prefer the default blue color.

<br />

<Showcase name="hero" />

## Installation

```sh
qwik-ui add radio-group
```

## Usage

```tsx
import { Label, RadioGroup, RadioGroupItem } from '@qwik-ui/styled';
```

```tsx
<RadioGroup>
  <div class="flex items-center space-x-2">
    <RadioGroupItem name="size" value="default" id="r1" />
    <Label for="r1">Default</Label>
  </div>
  <div class="flex items-center space-x-2">
    <RadioGroupItem name="size" value="comfortable" id="r2" />
    <Label for="r2">Comfortable</Label>
  </div>
  <div class="flex items-center space-x-2">
    <RadioGroupItem name="size" value="compact" id="r3" />
    <Label for="r3">Compact</Label>
  </div>
</RadioGroup>
```
